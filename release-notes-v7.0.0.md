## Bingo v7.0.0

### 词池可视化（全新）

以更形象、直观的方式查看词池——不再只是列表。

- **总览仪表盘**：词数 / 关系数 / 连通簇 / 孤立词等统计 + 关系类型分布条 + 图例
- **簇画廊**：按词数排序的连通分量卡片，点击进入对应区域
- **关系大图**：可拖动、可缩放的单 Canvas 全量渲染，中心辐射「辐射套辐射」树状布局，被辐射的点也继续辐射
- **极致性能**：三级 LOD（远景聚合点 / 中景点+边 / 近景词标签）+ 视口裁剪（只渲染屏内）+ 5 大类边合批，万词词池也能流畅拖缩
- **关系溯源**：点词查看释义与关联列表，点关系查看这条边的 AI 依据（reason / 例句 / 语域 / CEFR / 警示）
- **孤立词显隐**：一键切换只看有关系的词
- **入口**：词典页「词池 N 个」行 →「查看图谱」

### 头脑风暴背词（重大优化）

四个阶段重做选词、背词逻辑与视觉，让背词更易记。

- **选词精良**：质量门槛过滤低置信关系，按「关系强度 × 置信度 × 学习价值 × 类型权重」评分，沿最强键合贪心装配「学习簇」（易混词、反义、形近等优先成组出现）
- **记忆钩子**：揭示答案时展示当前词最强关联的「记忆联想」卡（关系 + AI 例句/理由）
- **拉钩复现**：评「重来」时把该词最强关联词拉到队首，趁热复现
- **主动回忆 MCQ（可选）**：对有同义/反义/易混/形近关联的词出选择题，干扰项来自会话内且排除真关联，答对计 Good、答错计 Again
- **视觉焕新**：当前词的迷你「辐射星图」（换词展开动画）、学习簇面包屑进度点、彩色「拼写+关系类型」关系卡、进度环式目标达成弹窗，目标设置加 50/100/200/300 快捷选项
- **可配置**：设置 →「头脑风暴背词」可调簇大小、质量阈值、主动回忆开关

### 修复

- **关系大图「只剩一个聚合点 + 缩放卡死」**：超大连通分量下首屏只显示一个远景聚合点、放大即 ANR 卡死——下调远景阈值使首屏默认呈现可读节点云，超预算时跳过画边消除卡死，并对准最大簇兜底
