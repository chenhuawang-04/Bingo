# Bingo v9.1.0

本版本新增完整考研英语组卷系统：从文章阅读页按题型收集并本地保存来源，按 `日期-第N套` 建立英语一/英语二试卷；9 个来源槽位凑齐后自动在后台生成题目和答案。轮换新题型使用单一、可配置的版本化槽位，不要求所有特殊题型同时出现。

题库现以试卷为第一层，支持查看来源与生成状态、失败续传、整卷顺序练习、草稿保存和进度恢复。后台生成具备事务写入、幂等恢复、单步骤重试、暂停/取消保护与答案完整性校验。Room 升级至 schema 38，云端题库备份升级至 schema 2，并兼容既有试卷、文章、作答记录和 v9.0.1 的完整单词展示功能。

完整说明见 `RELEASE_NOTES.md`。
