# Bingo v9.0.1

9.0.1 是 Bingo 9.0 的首个正式维护版本，重点统一并补全所有完整单词展示入口，修复同一个词在不同页面中信息不一致的问题。

## 完整且统一的单词详情

- 独立单词详情页、字典宽屏详情侧栏和背词答案页现在共享同一套详情组件与信息顺序。
- 完整展示拼写、音标、词义、构词拆解、词根解释、词形变化、近义词、形近词、同根词和自动关联词。
- 新增词簇详情：显示当前词所在的词簇、成员数量和其他成员，并可点击进入成员详情。
- 新增关系网络详情：显示邻接单词及同义、反义、同根、形近、易混淆等关系类型。
- 补齐词池、词组、词组释义、用法、标签和词组例句。
- 补齐从本地文章提取的例句，并继续支持 Cambridge 与 OED 云端例句切换。

## 背词体验

- 揭示答案后按需加载完整补充信息，不增加题面阶段的启动负担。
- 词簇编辑和关联背诵保持独立入口，避免与只读详情重复展示。
- 关联单词、词池成员和关系网络邻居均可打开对应词条。

## 宽屏与稳定性

- 修复字典宽屏侧栏漏传本地例句、云例句、词簇和关系网络的问题。
- 宽屏侧栏增加明确的单词标题，并维持基础信息/关联信息双栏布局。
- 快速切换单词时会取消旧详情和旧云例句请求，防止上一词的数据覆盖当前词。
- 加载新词时立即清空旧词簇、词池、关系、词组和例句，消除短暂串页。

## 兼容性

- Room schema 保持版本 37，无新增数据库迁移。
- 可直接从 v9.0.0 覆盖升级，现有单词、词簇、词池、文章、例句和学习记录不受影响。
