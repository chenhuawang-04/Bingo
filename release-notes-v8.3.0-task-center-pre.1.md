# Bingo v8.3.0-task-center-pre.1

## 任务与资源中心

- 统一前后台资源占用视图，后台调度会主动避让前台内存重型操作和数据库写入。
- 为持久化任务增加资源预算、优先级和稳定排序，兼顾用户操作响应与维护任务吞吐。
- 修复任务协程在登记前结束可能永久占用并发槽的竞态，并使用原子启动门闩避免重复初始化。
- 保留应用异常退出后的 `RUNNING` → `PENDING` 恢复机制，现有任务数据无需数据库迁移。

## 隐藏任务与更新检查

- 新增隐藏任务类别和 `APP_UPDATE_CHECK` 任务，后台任务页默认折叠隐藏任务，可手动展开查看。
- 每次启动按设置检查 `chenhuawang-04/Bingo` 的 GitHub Releases，并复用固定任务记录避免历史无限增长。
- 默认仅检查正式发布，严格排除 draft 和 prerelease；可在设置中开启测试版检查或关闭启动检查。
- 更新检查包含有限重试、语义化版本比较、检查结果持久化和 GitHub 发布页入口。

## 兼容性

- Room schema 保持版本 36，不新增或修改数据库列。
- 新任务类型继续按字符串序列化；旧版本任务和已有用户数据库可直接升级。
- 新设置使用 DataStore 默认值平滑生效：自动检查默认开启，测试版检查默认关闭。

## CI 验证

- GitHub Actions `CI`：JVM 单元测试与 Android 仪器测试。
- GitHub Actions `Android Release`：签名 APK/AAB 和 prerelease 发布。
